# SaaS智能比价平台

# 一、产品定位

## 1.1 产品背景

本产品是一套面向IVD行业集团型医疗企业与大中型供应链组织的SaaS智能比价平台，融合产品目录管理、价格比对、选品推荐、智能报价、分析决策等能力，支持平台端与多租户集团/组织端多层次业务需求。系统具备多组织、跨业务线、数据驱动与AI推荐的架构基础，支持业务流程数字化、决策智能化和全周期可追溯。

## 1.2 目标与定位

*   提供统一产品与价格比对平台，解决集团采购、供应商评估、合规管理等核心难题；
    
*   聚合集团/组织/供应商数据资产，支持高效比价、推荐、报价、决策全流程闭环；
    
*   支持定制化、多维度业务规则、灵活接入集团级、组织级和行业级数据；
    
*   对标国际一流SaaS比价、选品推荐平台，服务多类型医疗采购决策场景。
    

## 1.3 产品边界

*   平台端聚焦数据集成、模型管理、指标规则、模板共享与授权管理，不直接参与租户业务操作；
    
*   租户端支持集团（管理端）与业务（用户端）多组织独立运营、业务隔离；
    
*   支持多组织/多业务线数据融合、权限分隔、合规审计与自动化数据流转。

# 三、业务主流程与场景

## 3.1 主流程架构

1.  平台端统一集成各类主数据、产品目录、设备/试剂分类与注册证档案；
    
2.  平台端配置数据集、数据模型、指标体系、比价规则与分析模板；
    
3.  租户端（集团/组织/用户）通过授权获取平台分析模板、指标、报表等资源；
    
4.  租户端根据业务需求导入/录入采购清单，自动触发产品推荐与比价流程；
    
5.  系统智能比对历史价格、协议价、供应商与合规信息，生成多套选品与报价方案；
    
6.  方案比对、审批、输出，最终推动业务采购或策略调整；
    
7.  全流程日志与数据可溯源，支持风险预警、决策复盘。
    

## 3.2 典型场景

*   集团采购统一比价、同类产品替换、降本与协议评估
    
*   集团/组织自定义比价指标、策略规则配置
    
*   医疗集团内多医院/科室批量需求导入、分组智能推荐
    
*   报价、审批流集成与合规分析
    
*   跨业务线/多维度数据分析报表自动化推送

# 五、关键用户故事

1.  **集团采购经理**希望能按产品、项目、品牌等多条件批量导入采购需求，系统自动生成多套选品/替代/报价方案，支持比价与审批全流程闭环。
    
2.  **分析员/决策人**可以自定义分析维度与指标，动态组合报表，监控降本进度与合规风险，并能快速定位异常价格与供应商。
    
3.  **IT管理员/产品经理**能通过平台端配置共享模型、规则和报表模板，授权租户端灵活启用，所有操作均可审计、追溯。
    
4.  **租户集团管理员**希望在自身组织架构下，管理/分配指标、权限，定制符合本地管理需求的业务流程和报表视图。

# 六、技术栈与准备

## 6.1 技术架构

*   后端：Java（Spring Boot）、MyBatis Plus、Dubbo分布式服务
    
*   前端：React + Vite + TypeScript + Ant Design
    
*   数据库：MySQL、PostgreSQL等关系型数据库
    
*   大数据/分析引擎：可选~~Spark、Flink、BI~~SlectDB组件对接
    
*   权限/身份认证：OAuth2、JWT、租户多组织安全隔离
    
*   中间件：Redis、消息队列（Kafka/RocketMQ等）
    

## 6.2 集成能力

*   API开放平台，支持与ERP、SRM、DMS等系统集成
    
*   报表平台对接Antv S2、FineReport等
    
*   AI智能推荐引擎、规则引擎、数据同步中台可拓展
    

## 6.3 安全与合规

*   数据全链路加密、分级权限、操作审计
    
*   业务规则灵活配置，支持合规追溯与风险预警
    
*   支持主数据同步、组织与租户粒度的精细授权
